# recon3d

Paquete para flujos de visión estéreo y reconstrucción 3D.

Incluye módulos para:
- Calibración mono/estéreo
- Rectificación y mapas
- Cálculo de disparidad
- PnP y reproyección
- Fusión y medición
- Utilidades de E/S y visualización
- CLI simple para tareas de humo

## Instalación (editable)

```bash
pip install -e .
```

## Uso rápido

```bash
recon3d --version
recon3d smoke
```
